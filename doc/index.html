NAntContrib : Extra goodness for NAnt

NAntContrib

NAntContrib is the project for tasks and tools that haven't made it into the main NAnt distribution yet or for whatever reason don't belong there. Don't know what NAnt is ? then you need to go here.

Its also the home of the Slingshot buildfile generator and the visual studio.Net addin for NAnt.

Getting Started

Download the latest binaries as part of the lastest NAnt release here.

Daily builds are currently unavailable but we're working on it.

Read the NAnt Contrib Task Documentation  or explore the NAnt Wiki

 Read and search the mail archives: NAntContrib-Dev

Get the latest cvs tree.

 Want to contribute ? or submit patches ? make sure you have the latest source then submit your proposal or fix to the developer list.
