NAntContrib Release Notes

0.84 (December 26, 2003)

Task changes

<regasm>
  • Fixed issue when long filenames were specified for the typelib or regfile attribute.

0.84-rc 2 (December 21, 2003)

No changes

0.84-rc 1 (December 6, 2003)

New Tasks

<vssdif>
  • Shows all changes to a Visual SourceSafe project after a specified label.

Task changes

<msi> and <msm>
  • For entries in the File table, the language will now be supplied.
  • Nullable columns for a custom table are now properly supported.
  • Added checkinterop attributes to the component and fileoverride elements.
  • Support pure registry components.
  • Improved error reporting.
<slingshot>
  • Fixed bug with regards to parameters.
  • Added support for Visual Studio.NET 2003 solutions.
<sql>
  • Made task less chatty.
<validatexml>
  • Renamed schemas child element name from <schemaref> to <schema>.
<version>
  • Improved error reporting and documentation.
<vssget>
  • Added removedeleted attribute to support removal of local files that were deleted files from the Visual SourceSafe repository.
  • Added usemodtime attribute to support setting the timestamp of local files to the modification time of these files in the Visual SourceSafe repository.
  • Added unit tests.
