nant Addin for Visual Studio.NET

Task Nodes

The Script Explorer displays a Task node for each NAnt task below each Target node in an NAnt script. Use the toolbar, Properties window, and pop-up menu to edit the properties of and invoke commands on a Task node.

The following commands are supported when a Task node is selected:

Cut
Copy
Delete
Move Up
Move Down

See the NAnt Task documentation for the list of properties supported by the addin.

Note: Each NAnt Task node has a unique icon and name. NAnt Tasks are the units of work performed by a Target and have different properties depending on the Task. Consult the NAnt Task reference for a list of all supported properties.
