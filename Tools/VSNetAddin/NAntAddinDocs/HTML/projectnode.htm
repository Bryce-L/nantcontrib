NAnt Addin  

nant Addin for Visual Studio.NET

Project Nodes

The Script Explorer displays a Project node for each open NAnt script (.build file) in the projects open in the Visual Studio.NET IDE. Use the toolbar, Properties window, and pop-up menu to edit the properties of and invoke commands on a Project node.

The following commands are supported when a Project node is selected:

Build Project
View Code
Add Property
Add Target
Paste

See the NAnt Project documentation for the list of properties supported by the addin.
