nant Addin for Visual Studio.NET

View Code

When a Project node is selected in the Script Explorer window, the addin's pop-up menu or toolbar can be used to view the NAnt script's XML source code. Whenever you view the source code of an NAnt script, the Project node for that script is collapsed in the Script Explorer until you are done editing the source. If you attempt to expand the Project node while it is being edited, you will be given the option of closing the open source code before continuing.

If you cause an XML parsing error by writing XML that is not well-formed during an editing session, the Project node for the script being edited will display the line number and error in the Script Explorer when you close the source. You can then invoke the View Code command again to fix the error.
