Paste

When a node in the Script Explorer window has been cut or copied to the clipboard, it can be pasted into a new location in an NAnt script using the pop-up menu. The following table lists which nodes can be pasted at which locations:

 

Source Node Target Node
Property Project, Target
Target Project
Task Target

 

Note: Nodes can be pasted from one project to another.
