nant Addin for Visual Studio.NET

Properties Window

When any node in the Script Explorer is selected, the Visual Studio.NET Properties Window displays properties that can be used to edit the properties of the node. The Properties Window is displayed below the Script Explorer in the default IDE layout.

 


Figure 1: The Properties Window with a selected Project node.

 

Most properties in the addin are strings. NAnt uses the concept of a File Set to configure a set of files and/or directories affected by a task. Some Task Nodes display an array editor for the files in a File Set in the Properties Window when selected. To include or exclude files and/or directories from a File Set select the property and click the button on the right side of the property edit field. A dialog will appear to allow you to edit the individual collection of items.

 


Figure 2: The File Set editor dialog.

 

Once items have been added to a File Set, the property can also be expanded by selecting the property and clicking the expansion icon to edit the properties of an individual included or excluded file or directory.

 


Figure 3: The Properties Window with an expanded File Set.
