NAnt Addin  

nant Addin for Visual Studio.NET

Delete

When a Property node, Target node, or Task node is selected in the Script Explorer window, the addin's pop-up menu can be used to deleted from the script.

Caution: Once a node has been deleted from the script, it needs to be re-added from scratch, which can be painful if it is a Task node with many configured properties. There is no "Undo" function in the addin so please be sure you are deleting the correct node when you click "Yes" to the delete confirmation message.
