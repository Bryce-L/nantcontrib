nant Addin for Visual Studio.NET

Build Target

When a Target node is selected in the Script Explorer window, the addin's pop-up menu or toolbar can be used to Build a Target. The output from the build process is displayed in the Build Output Window. If an error occurs during a build that displays a file name and line number, the error can be clicked to load the XML source code for the NAnt script that caused the error. This will take you to the line in the script where the error ocurred.

You can only build one Target at a time, however if you have you have multiple Projects open, you can invoke the Build Target command on each Target Node to queue up the builds so they will execute one after another.
