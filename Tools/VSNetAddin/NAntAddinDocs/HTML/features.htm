NAnt Addin  

nant Addin for Visual Studio.NET

Summary

This Addin integrates the NAnt build system with Visual Studio.NET.

NAnt is an open source C# version of the popular Apache Ant build system written in Java.

The NAnt Addin works with any .build files that are part of projects you have open in Visual Studio.NET. The main user interface of the Addin is a TreeView that is designed to fit in the same set of tabbed windows that contains the Solution Explorer, Class View, Resource Editor, and Help Contents/Search in the Visual Studio.NET IDE.
