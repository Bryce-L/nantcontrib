nant Addin for Visual Studio.NET

Script Explorer

The main window through which the addin is used is the Script Explorer. This window is floating when you first load the NAnt addin. You can dock it into any of the windows in the Visual Studio.NET IDE, but it is designed to dock with the tabbed windows that also contain the Solution Explorer, Class View, Resource Editor, and Help Contents/Search in the default IDE layout.

The Script Explorer displays contains a Toolbar at the top that provides quick access to common functions. Below the Toolbar is a tree containing a node for each NAnt script in the projects open in Visual Studio.NET. This node is the Project node.

 


Figure 1: The Script Explorer with a single open Project node.

 

Right-click on any node in the Script Explorer's tree to display a pop-up menu. This menu provides the commands on the toolbar plus some additional functions depending on the node you have selected. You can select a node in the tree and hit the F1 function key to display online help for that node. If the Visual Studio.NET Dynamic Help window is also visible, links to help topics relevant to the node will also be displayed.
